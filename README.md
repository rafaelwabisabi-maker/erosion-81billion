# Erosion — 81 Billion

An interactive data-art project visualizing the 81 billion tons of topsoil lost globally each year.

## About

Erosion translates an invisible ecological crisis into a sensory experience. Through generative visuals and real data, it makes the scale of annual topsoil loss tangible — connecting viewers to the ground beneath them. Submitted to Prix Ars Electronica 2026.

## Tech Stack

- HTML / CSS / JavaScript
- Generative visuals
- Data visualization

Submitted to Prix Ars Electronica 2026.
